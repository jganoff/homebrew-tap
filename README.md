# homebrew-tap

Custom [Homebrew](https://brew.sh) tap.

## Usage

```sh
brew tap jganoff/tap
brew install jganoff/tap/<formula>
```

## Formulae

| Formula | Description |
|---------|-------------|
| [wsp](https://github.com/jganoff/wsp) | Multi-repo workspace manager using local git clones |

### wsp

```sh
brew install jganoff/tap/wsp
```

Supported platforms: macOS (Apple Silicon), Linux (ARM64, x86_64).
